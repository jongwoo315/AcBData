## $within 메서드


```python
# 정의한 범위(사각) 내에 있는 좌표를 출력
box = [[40, -40], [-40, 40]]
users_in_boundary = db.users.find({'user_location':{'$within':{'$box':box}}})
users_in_boundary
```




    <pymongo.cursor.Cursor at 0x7f42eff2d550>


<br>

```python
for a in users_in_boundary:
    print a['username'], a['user_location']
```
